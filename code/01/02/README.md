- muutuja
- muutuja deklareerimine

  - const
  - prompt()
  - alert()
  - confirm()
  - console.log()
- Muutujate tuubid
  - undefined
  - null

- Operatoorid

  - Maaramisoperaatorid (=, *=, +=, -=)
